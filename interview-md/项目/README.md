      代码初始化，可以封装成脚手架，类似create-vite, 可以内置上面说的各种规范，新项目直接启动

      代码联调效率， 比如接口json自动生成Typescript接口类型等等，比如mock数据工具
      代码上线效率，发布部署，部署结果同步到聊天群等等，把日常重复的行为自动化
